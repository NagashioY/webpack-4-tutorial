解説記事はこちらをご覧ください。

[webpack 4 入門 ](https://qiita.com/soarflat/items/28bf799f7e0335b68186)

### インストール
```
npm install
```

or

```
yarn install
```

### 使い方

#### `webpack --mode development --watch`を実行したい場合

```
npm run dev
```

or

```
yarn run dev
```

#### `webpack --mode production`を実行したい場合

```
npm run build
```

or

```
yarn run build
```
