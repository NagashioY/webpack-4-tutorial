# webpack-4-tutorial

### インストール
```
npm install
```

or

```
yarn install
```

### 使い方

#### `webpack --mode development --watch`を実行したい場合

```
npm run dev
```

or

```
yarn run dev
```

#### `webpack --mode production`を実行したい場合

```
npm run build
```

or

```
yarn run build
```
